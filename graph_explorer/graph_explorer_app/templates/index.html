Workspaces

Select a workspace from the dropdown. Click + Add to create a new workspace, choose a data plugin and visualizer, then click Create.

Filtering & Search

Use Filter Query to apply conditions using simple syntax (e.g., "age > 30", "name == John", "height >= 175"). Use Search to look across all fields. Applied filters appear below the toolbar, and can be removed anytime.

CLI Usage

Click the CLI button to open the command line interface. Type commands (like list nodes, filter field=value) and press Enter. Use Clear to reset the log.
